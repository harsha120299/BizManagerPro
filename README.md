## ⚙️ How to Run

### 1. Clone the repo

```bash
git clone https://github.com/username/BizManagerPro.git
cd BizManagerPro
```

### 2. Start Backend

```bash
cd BackEnd
mvn clean install
mvn spring-boot:run
```

### 3. Start Frontend

```bash
cd ../FrontEnd
# If React
npm install
npm start
# If plain HTML/JS just open index.html in browser
```

### 4. Open in Browser

* Backend runs on: `http://localhost:8080`
* Frontend runs on: `http://localhost:3000` (if React)
